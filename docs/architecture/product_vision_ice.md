# 🌌 产品迭代愿景：从“被动控制”到“无感智能” (Zero-UI Platform)

> **“我们不是在制造一个万能遥控器。我们在打造一种‘预判你所想，自然如呼吸’的平台级生活方式。”**

## 1. 哲学重构：推倒重来
我们将把原有的“传统智能设备伴侣”的控制面板模式彻底推倒重来。那种充满复杂菜单、开关列表和设置项的设计太传统、太工业化了。从今天起，我们将把这种极致体验抽象为平台通用能力 —— **“无感智能 (Zero-UI) 平台”**。

我们的核心哲学是：**App 的存在，是为了让用户可以忘记它的存在。** 真正的智能不需要用户去“控制”，而是主动“预知”并“迎合”用户的生活节奏。无论是制冰机、咖啡机、还是全屋灯光，都应遵循这一哲学。

---

## 2. 平台级核心能力 (Platform Pillars)

### 2.1 视觉与交互：极简主义的物理映射 (Physics-Based UI Engine)
*   **摒弃传统 UI 组件**：干掉所有的汉堡菜单、开关 Toggle、Tab 栏。将操作界面抽象为“物品状态的物理映射”。
*   **平台化 3D/动画渲染器**：
    *   *制冰机场景*：屏幕上只有极其逼真的、基于物理引擎的冰块与玻璃杯动画。
    *   *咖啡机场景*：屏幕上是萃取油脂的流体动画与香气氤氲的视觉粒子。
    *   *灯光场景*：屏幕是根据一天光照色温实时变化的渐变光晕。
*   **直觉式手势抽象**：将繁琐的“设置参数 -> 点击开始”转化为符合人类直觉的单一手势。例如：划入冰块即开始制冰；轻触光晕即点亮房间。

### 2.2 本地 AI 核心：跨设备预知引擎 (Precognitive Intelligence)
*   **从“响应指令”到“预判需求”**：一个还需要你告诉它何时工作的“智能”设备，一点都不智能。
*   **全局习惯学习模型 (Federated Habit Engine)**：利用我们的端侧 AI 与 Isar 数据库，跨设备记录用户的行为节律，构建用户的生活图谱。
    *   *场景示例*：系统学习到你“每周五晚上 7 点会喝一杯加冰威士忌，并伴随柔和的灯光”。当周五 6:45 你走进家门（触发地理围栏）时，完美的冰块已经备好，氛围灯已调至暖色。
*   **零打扰后台编排**：无需每次打开 App，端侧 Agent 会在后台基于时间、温度、历史习惯自主编排多设备联动策略并下发指令。

### 2.3 感官体验：多模态 ASMR 反馈库 (Sensory Design System)
*   **拒绝工业电子音**：彻底废弃廉价的、系统默认的“叮”声通知。建立平台级的**高保真 ASMR 音效与触觉反馈库**。
*   **设备专属感官闭环**：
    *   *制冰完成*：冰块落入玻璃杯的清脆撞击声 + 手机高级线性马达的干脆震动。
    *   *咖啡萃取完成*：咖啡滴落的浓郁液滴声 + 绵密的脉冲震动。
    让用户的听觉和触觉成为确认设备状态的自然延伸。

## 3. 技术架构对齐 (Alignment with Current Architecture)

这个极其浪漫且极客的平台愿景，完美契合我们当前的**“端侧大模型 + Agent”**底层架构：

1.  **预知能力的基石 (Isar DB)**：我们现有的 `BehaviorLog` (行为日志) 不仅用于单设备 RAG 问答，将升级为**跨设备习惯预测模型**的数据源，完全在本地计算，绝对保护隐私。
2.  **通用物理渲染管道 (Flutter Impeller)**：利用 Flutter 强大的跨平台渲染能力，构建可复用的 Shader 与骨骼动画池，供不同 IoT 设备按需加载物理映射视觉。
3.  **统一异步调度器 (Intent Executor)**：基于端云协同架构，设备影子 (Device Shadow) 不再只反映状态，而是作为 Agent 预判执行的“数字孪生靶场”，实现跨局域网的静默控制。

## 4. 平台化演进路线 (Next Steps)
- [ ] **Zero-UI 渲染框架封装**：将现有的 Flutter 视图层重构为支持动态注入 3D 模型/Shader 动画的物理映射容器。
- [ ] **通用手势映射层**：开发一套将“划动/长按/重力感应”等直觉手势自动映射为 JSON 控制指令的中间件。
- [ ] **端侧预判算法组件化**：在端侧 Agent 中抽离出基于时间序列的“多设备习惯预测模块 (Habit Prediction Module)”，提供标准 API 供各类设备接入。
- [ ] **多模态反馈资源池构建**：建立包含 ASMR 音效与 CoreHaptics 震动参数的标准化感官资源包。
