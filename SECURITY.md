# Security Policy

## Supported Versions

Currently, the `main` branch is the only actively supported version for security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

As this project focuses on **Smart Home** and **Privacy**, we take security issues very seriously.

If you discover a security vulnerability within this project (such as a flaw in the Edge-Cloud isolation, Isar DB encryption bypass, or Prompt Injection leading to unauthorized hardware control), please DO NOT open a public issue.

Instead, please send an email directly to the maintainers (or open a private security advisory if enabled on GitHub). 

We will strive to acknowledge your report within 48 hours and provide a timeline for addressing the issue.
